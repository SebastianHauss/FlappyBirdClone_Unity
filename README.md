### Installation and Running

1. **Clone the Repository**
   ```bash
   git clone https://github.com/SebastianHauss/FlappyBirdClone_Unity.git
   cd FlappyBirdClone_Unity
   ```

   - Launch Unity Hub

   - In Unity, click the Play button to start
